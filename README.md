# Что может seller.py
## Поможет поменять цены на товар с маркетплейса OZON

1) Получаем список необходимых товаров с указанием:
* Наименования товара
* Артикула
* Текущей цены

2) Получаем список остатка товаров


3) При желании можем поменять цену на товар
